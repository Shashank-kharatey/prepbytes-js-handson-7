Q.2 Write a program which takes a numebr N as input from the user and You need to reverse the number. Note: You have to complete Reverse_Number function. No need to take any input.

Ans.2 var Reverse_Number = (N) => 
        {
          var L=0;
          while(N)
          {
            L+=N%10;
            N=parseInt(N/10);
            if (N!==0)
            {
              L*=10;
            }
          }
          return L;
        };

Q.4 You are given a string S, and your task is to return the length of the string S. Note: You have to solve it without using length method. You have to complete findLength function. No need to take any input.

Ans.4 const findLength = (S) => 
        {
          var i=0;
          var count=0;
          while (S.charAt(i))
          {
            count++;
            i++;
          }
          return count;
        };

Q.5 You are given a string S consisting of two letters A and D,where each character represent the winner of N games played between Aditya and Danish, where letter A represents the win of Aditya and letter D represents the win of Danish. You need to find out the that which player wins the maximum number of games or there is a draw between them. Note: You have to complete Game_Winner function. No need to take any input.

Ans.5 var Game_Winner = (S) => 
        {
          var countA=0;
          var countD=0;
          for (i=0; i < S.length; i++)
          {
            if (S.charAt(i)==='A')
            {
              countA++;
            }
            else
            {
              countD++;
            }
          }
          if (countA>countD)
          {
            return "Aditya";
          }
          else if (countD>countA)
          {
            return "Danish";
          }
          else 
          {
            return "Draw";
          }
        }
